Workflow

How it works

  1. Upload an image or PDF document.
  2. The system classifies the document type.
  3. If it is an invoice, the invoice fields are extracted.

Classification

Stacking ensemble structure

  • Base learners: SVD + SVM, HOG + SVM, HOG + Random Forest, and CNN analyze the same document independently.
  • Meta-features: each model produces class probabilities, which are combined into a single feature vector.
  • Meta-classifier: a Logistic Regression model learns from the combined outputs.
  • Final prediction: the ensemble selects the class with the strongest combined evidence.
