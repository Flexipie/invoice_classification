To: Dr, Robert L. Suber From: Joel A. Seckar

1 requested that Mike Morton review the proposed statistical approach Pharmaco-LSR
plans to use for the 12-month study of methyl bromide fumigated feed in the dog. His
responses led to the following comments for Pharmaco-LSR's statisticians:

+ Since tests on the variance are not very powerful for small sample sizes, we would
recommend running Bartletts test at the .05 level rather than the .01 level, The
protocol states that Bartlett's testis being run as a two-sided test, We assume that this
is a misstatement. We presume that Pharmaco-LSR will not reject the null hypothesis
if the Bartlet test statistic is too small!

+ Ifthe variances are unequal (significant Bartlet’s test), itis true that the ANOVA
assumptions are violated, but s0 are the assumptions of the Kruskal-Wallis test (it also
assumes a common vatiance, though it allows for a non-normal error distribution; for
instance, see p. 115 of Hollander and Wolfe). If Bartlett's test shows unequal
variances, we would recommend transforming the data rather than using the non-
parametric set of tests. Presumably either a square root or a logarithmic
transformation would mitigate the heterogeneity of variance,

«In examining the data for trend, we would test the data using a linear contrast with the
anova error term. This is equivalent to testing the mean square from the regression
against the pure error term that is used to test for lack of fit, The straight regression
includes the lack of fit 1erm in addition to the pure error term as part of the error---if
there is curvature, this will inflate the error term, Ifitis desired to test if there is
curvature, a test of the quadcatic term vs. the pure error term can be run (or, for that
matter, the standard lack of fit test can be run).
